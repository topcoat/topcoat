    Scrolling example for ios & android
    w/ overflow:scroll & -webkit-overflow-scrolling: touch; for iOS
    android has no overflow:scroll
    On android I implemented my own version of scrolling using css transform. The menu on the left is without it the menu on the right has the scrolling implemented. Usually a menu on an android 2.x would be fully expanded because overflow:scroll == overflow:hidden on android. With this implementation you can hide the overflow, listen on touchevents and move the content with css transform
    on ios its regular css, overflow:scroll
    I noticed that using `native scrolling` on menus breaks the app (makes the div flexible, you can pull it from any direction and it doesn't feel like a native app) I did however use native scrolling on the main content area (without it overflow:scroll does not have the inertia)
